## 📚 Descrição

App To Do List, desenvolvido em React.js & TypeScript.

## 🏃 Rodando o aplicativo

```bash
# Clone o repositório:
git clone git@github.com:victorozoterio/to-do-list-frontend.git

# Execute o comando:
npm run dev

# Abra a aplicação acessando a seguinte URL:
http://localhost:5173/
```
